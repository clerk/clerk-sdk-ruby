# OrganizationDomainStatus

Status of the verification. It can be `unverified` or `verified`

## Example Usage

```ruby
require "clerk-sdk-ruby"

value = OrganizationDomainStatus::UNVERIFIED
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `UNVERIFIED` | unverified   |
| `VERIFIED`   | verified     |
