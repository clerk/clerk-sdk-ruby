# ListWaitlistEntriesStatus

Filter waitlist entries by their status

## Example Usage

```ruby
require "clerk-sdk-ruby"

value = ListWaitlistEntriesStatus::PENDING
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `PENDING`   | pending     |
| `INVITED`   | invited     |
| `COMPLETED` | completed   |
| `REJECTED`  | rejected    |
