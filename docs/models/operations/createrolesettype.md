# CreateRoleSetType

The type of the role set. "initial" role sets are the default for new organizations.
Only one role set can be "initial" per instance.

## Example Usage

```ruby
require "clerk-sdk-ruby"

value = CreateRoleSetType::INITIAL
```


## Values

| Name      | Value     |
| --------- | --------- |
| `INITIAL` | initial   |
| `CUSTOM`  | custom    |
